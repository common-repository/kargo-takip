Müşterilerinizin kargolarını takip etmesine olanak sağlayan bir kargo takip eklentisidir. Kargo takip eklentisi aras kargo, mng kargo , sürat kargo ve yurt içi kargo  gibi çok sık kullanılan kargo firmalarının kargo takibini yapmanıza olanak sağlayan kullanımı kolay bir eklentidir. Eklenti ile hem üye sipariş detaylarından kargo takibini yapabilmekte hem de admin panelinden yöneticiler kargo durumunu görüntüleyebilmektedir.

== Installation ==
Eklentinin çalışabilmesi için woocommerce eklentisi gereklidir.

* Eklentiler > Yeni Ekle linkini tıklayarak eklentiyi kurun
* Eklentiler > Yüklü eklentiler  menüsü altından aktif edin
* Ayarlar > Kargo Takip menüsü altından api bilgilerinizi girip kullanmaya başalayın 

Eklenti klasöründe  Readme.docx dosyası içerisinde görsel kurulum bilgileri detaylı olarak anlatılmıştır.
